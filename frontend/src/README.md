# Inventory & Order Management System

A full-stack Inventory and Order Management application built with React, Node.js, Express, PostgreSQL, and Neon Database.

## Features

- Add and manage products
- Add and manage customers
- Create orders
- Automatic stock reduction
- Insufficient stock validation
- Product, customer, and order listing
- Responsive dashboard UI

## Tech Stack

Frontend: React, Vite, CSS  
Backend: Node.js, Express.js  
Database: PostgreSQL  
Cloud Database: Neon  
API: REST API
